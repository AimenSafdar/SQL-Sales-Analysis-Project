# Sales Analysis Project

**Description:**  
This project demonstrates a complete sales analysis using PostgreSQL. It includes creating tables, inserting data, performing joins and aggregations to generate business insights.

**Tools:** PostgreSQL, pgAdmin

**Dataset:**  
- Customers  
- Products  
- Orders  
- Order Items

**Analysis Queries:**  
- Total Revenue  
- Top Selling Products  
- Top Customers  
- Revenue by Country  
- Monthly Sales Trend  
- Product Revenue Ranking
